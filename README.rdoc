= Phoney

Deal with phone numbers in Ruby. Special integrations with Rails.

= A Conversation with Phoney

    >> p = Phoney::PhoneNumber.new('+1 555.312.1990')
    => #<Phoney::PhoneNumber:0x101536bf8 @digits="15553121990">
    >> p.to_s
    => "+1 (555) 5312199"
    >> p.to_s(:db)
    => "15553121990"
    >> p.area_code
    => "555"
    >> p.country_code
    => "1"
    >> p.number
    => "5312199"
